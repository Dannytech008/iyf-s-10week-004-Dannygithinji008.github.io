## ✨ Features

- ✅ **Basic Arithmetic Operations**: Addition (+), Subtraction (-), Multiplication (*), Division (/), and Modulo (%)
- ✅ **Clean & Intuitive UI**: Simple design with easy-to-read display
- ✅ **Real-time Calculation**: Instant results when operation completes
- ✅ **Number Input**: Support for multi-digit numbers
- ✅ **Decimal Point Support**: Calculate with decimal numbers
- ✅ **Error Handling**: Handles division by zero and invalid operations
- ✅ **Keyboard Friendly**: Can be used with mouse clicks
- ✅ **Lightweight**: No external dependencies required
- ✅ **Responsive Design**: Works across different screen sizes
